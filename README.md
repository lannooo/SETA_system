# SETA_system
该教学辅助管理系统是一个课程项目，提供的功能包括：
- 辅助教师完成教学工作(包括作业管理、教学资源管理、在线答疑等）
- 帮助学生在线参与课程学习(包括完成作业、资源下载、学习交流等)
- 同时支持非课程内的学生访问部分内容
- 管理员进行课程管理。
